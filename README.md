# dust-so-example

A sample solution for [a StackOverflow question](http://stackoverflow.com/questions/25365396/load-compiled-dust-js-templates-in-node-js) on how to use dust templates in a particular way.

### Instructions

1. `npm i`
2. `npm start`
3. visit [http://localhost:8000](http://localhost:8000) and [http://localhost:8000/pre](http://localhost:8000/pre)
4. visit [http://localhost:8000/gist](http://localhost:8000/gist) and [http://localhost:8000/gist/pre](http://localhost:8000/gist/pre)
